# Mini Game Zone 🎮

**Mini Game Zone 🎮** is a collection of 10 fun browser-based games built with **HTML, CSS, and JavaScript**.  

Play classics like **Tic-Tac-Toe** and **Rock Paper Scissors**, while improving **typing speed, accuracy, focus, and decision-making**.  

✨ Easy to use, **open-source**, customizable, and great for both fun and learning!
