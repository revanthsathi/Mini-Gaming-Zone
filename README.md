# 🎮 Mini Game Zone

A fun and interactive collection of browser-based mini games developed using **HTML, CSS, and JavaScript**. This project combines multiple classic games into a single responsive website, providing an enjoyable gaming experience while demonstrating front-end web development skills.

---

## ✨ Features

- 🎲 Multiple games in one website
- 🎨 Clean and user-friendly interface
- 📱 Responsive design
- ⚡ Fast and lightweight
- 🌐 Runs directly in any modern web browser
- 🖱️ No installation required

---

## 🚀 How to Run

1. Clone this repository:

```bash
git clone https://github.com/YOUR_USERNAME/Mini-Game-Zone.git
```

2. Open the project folder.

3. Double-click **index.html**  
   **OR**

```bash
start index.html
```

4. Select any game from the homepage and start playing.
